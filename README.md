# AppLog
Printing Logs in File and present in viewController.

## Usage

```Swift
AppLog("Hey error occured")
```

Output:
```
*-*-*-*-*-*-*-*-*-*-*-*-*-*-*-*-*
⏰ Date: 2019-11-30 10:10:10 +0000
---------------------------------
📄FileName: ~/AppName/AppDelegate.swift
---------------------------------
🔨Function: fetchResource(_:)
---------------------------------
♦️LineNumber: 78
---------------------------------
👨🏻‍💻 Message: Hey error Occured
---------------------------------
*-*-*-*-*-*-*-*-*-*-*-*-*-*-*-*-*
```
